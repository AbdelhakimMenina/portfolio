# Icônes

Ce répertoire contient les icônes SVG des compétences et outils.

## Structure

Les icônes doivent être nommées selon le nom de la compétence en minuscules, avec les espaces remplacés par des tirets et les points supprimés.

## Exemples de noms de fichiers

- `reactjs.svg` pour "React.js"
- `nodejs.svg` pour "Node.js"
- `gitlab-cicd.svg` pour "GitLab CI/CD"
- `postgresql.svg` pour "PostgreSQL"
- `vuejs.svg` pour "Vue.js"
- `expressjs.svg` pour "Express.js"
- `typescript.svg` pour "TypeScript"
- `javascript.svg` pour "JavaScript"
- `docker.svg` pour "Docker"
- `aws.svg` pour "AWS"
- etc.

## Format recommandé

- Format : SVG (recommandé pour la qualité et la scalabilité)
- Taille : 24x24px à 48x48px
- Style : Monochrome ou avec couleurs selon votre préférence

## Note

Si une icône n'existe pas, un placeholder "Icon" sera affiché automatiquement.
